Webpack ж·ұе…Ҙпјҡд»Јз ҒеҲҶеүІгҖҒsplitChunks дёҺ Runtime Chunk

еҸ‘иЎЁдәҺ2026-04-25|жӣҙж–°дәҺ2026-04-21|е·ҘзЁӢеҢ–
|йҳ…иҜ»йҮҸ:

д»Јз ҒеҲҶеүІеҶіе®ҡ йҰ–еұҸдёӢиҪҪеӨҡе°‘гҖҒзј“еӯҳе‘ҪдёӯзҺҮ дёҺ е№¶иЎҢиҜ·жұӮж•°гҖӮWebpack 5 йҮҢз”ұ import() дә§з”ҹ async chunkпјҢз”ұ splitChunks жҠҪ vendor / commonпјҢ**runtimeChunk** еҲҷжҠҠ manifest д»ҺдёҡеҠЎ chunk йҮҢжӢҶеҮәпјҢйҒҝе…Қ дёҡеҠЎж”№дёҖиЎҢ вҶ’ vendor hash е…ЁеҸҳгҖӮдёӢйқўеҶҷ зӯ–з•ҘгҖҒй»ҳи®ӨиЎҢдёәгҖҒй…ҚзҪ®зӨәдҫӢдёҺжҺ’йҡңгҖӮ

дёҖгҖҒдёүз§Қ chunk еҝғжҷәжЁЎеһӢ

зұ»еһӢ жқҘжәҗ е…ёеһӢеҶ…е®№
Entry chunk entry е…ҘеҸЈ + еҗҢжӯҘдҫқиө–
Async chunk еҠЁжҖҒ import() и·Ҝз”ұйЎөгҖҒйҮҚеһӢз»„д»¶
Split chunk splitChunks е‘Ҫдёӯ node_modulesгҖҒиў«еӨҡе…ҘеҸЈе…ұдә«жЁЎеқ—

дәҢгҖҒimport() дёҺйӯ”жі•жіЁйҮҠ

  • **webpackChunkName**пјҡеҗҲе№¶еҲ°еҗҢдёҖ е‘ҪеҗҚ chunkпјҲй…ҚеҗҲйӯ”жі•жіЁйҮҠеҲҶз»„пјүгҖӮ
  • webpackPrefetch / webpackPreloadпјҡжҸ’е…Ҙ <link rel="prefetch/preload"> жҸҗзӨәпјҲд»ҚеҸ—жөҸи§ҲеҷЁзӯ–з•ҘеҪұе“ҚпјүгҖӮ

дёүгҖҒsplitChunks ж ёеҝғеӯ—ж®өпјҲWebpack 5пјү

**chunks: 'all'**пјҡеҗҢжӯҘе…ҘеҸЈйҮҢ д№ҹиғҪ жӢҶеҮә vendorпјҲй»ҳи®ӨеҸӘеҜ№ async иҫғдҝқе®ҲпјҢйңҖжҳҫејҸпјүгҖӮ
**cacheGroups.priority**пјҡж•°еӯ— еӨ§ дјҳе…ҲеҢ№й…ҚгҖӮ

еӣӣгҖҒдёәд»Җд№ҲиҰҒ runtimeChunk: 'single'

Webpack иҝҗиЎҢж—¶д»Јз ҒпјҲ__webpack_require__гҖҒchunk еҠ иҪҪиЎЁпјүй»ҳи®ӨеҸҜиғҪ жү“иҝӣ entryгҖӮ
дҪ еҸӘж”№ дёҡеҠЎжЁЎеқ— ж—¶пјҢmodule id жҳ е°„ еҸҳеҢ–дјҡеҜјиҮҙ еҗ« runtime зҡ„ chunk hash еҸҳпјҢй•ҝжңҹзј“еӯҳзҡ„ vendor д№ҹеҸҜиғҪи·ҹзқҖеҸҳгҖӮ

жӢҶеҮә runtime еҗҺпјҡдёҡеҠЎ hash еҸҳ еҸӘеҪұе“Қ дёҡеҠЎ chunk + runtimeпјҢvendor жӣҙзЁіе®ҡгҖӮ

еӨҡйЎөеә”з”Ёпјҡжңүж—¶з”Ё multiple жҲ–жҢүе…ҘеҸЈжӢҶпјҢйҒҝе…Қ еҚ•ж–Үд»¶ runtime иҝҮеӨ§пјӣйңҖжқғиЎЎгҖӮ

дә”гҖҒзӨәдҫӢпјҡгҖҢжҠҠ UI еә“еҚ•зӢ¬жү“дёҖеҢ…гҖҚ

жіЁж„ҸпјҡиҝҮеәҰжӢҶеҲҶ вҶ’ HTTP/1.1 йҳҹеӨҙйҳ»еЎһгҖҒжҸЎжүӢеӨҡпјӣHTTP/2 д»ҚдёҚе®ң еҮ еҚҒдёӘе°Ҹж–Үд»¶пјҲеҺӢзј©ж•ҲзҺҮгҖҒжңҚеҠЎеҷЁжҺЁйҖҒе·ІејҸеҫ®пјүгҖӮдёҖиҲ¬д»Ҙ дҪ“з§ҜжӣІзәҝ + Lighthouse дёәеҮҶгҖӮ

е…ӯгҖҒжҺ’йҡңпјҡchunk еҠ иҪҪйЎәеәҸй”ҷгҖҒз©әзҷҪйЎө

  • manifest дёҺдёҡеҠЎ chunk дёҚеҢ№й…Қпјҡеёёи§ҒдәҺ CDN зј“еӯҳ еҸӘеҲ·ж–°йғЁеҲҶж–Үд»¶ вҶ’ ејәзј“еӯҳзӯ–з•Ҙ еә”еҜ№ е…ЁйҮҸ hash дёҖиҮҙгҖӮ
  • chunkLoadErrorпјҡзҪ‘з»ңж–ӯгҖҒй”ҷиҜҜзүҲжң¬гҖҒи·Ёеҹҹ crossOrigin и„ҡжң¬жӢҝдёҚеҲ° иҜҰз»Ҷй”ҷиҜҜгҖӮ

дёғгҖҒж”¶жқҹ

д»Јз ҒеҲҶеүІ = import() еҲ’з•Ң + splitChunks жҠҪе…¬е…ұ + runtime дҝқзј“еӯҳгҖӮе…Ҳ йҮҸдҪ“з§ҜдёҺиҜ·жұӮж•°пјҢеҶҚеҫ®и°ғ cacheGroupsпјҢйҒҝе…Қ дёәжӢҶиҖҢжӢҶгҖӮ
