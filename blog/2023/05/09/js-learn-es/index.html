es6+部分方法

数组打平方法

递归
用falt新方法

了 Array.prototype.flat(),返回一个对要打平 Array 实例的浅复制副本
falt 默认参数1 表示展开的层

Array.prototype.flatMap()

arr.flatMap(f)与 arr.map(f).flat()等价

Object.fromEntries()

用于通过键/值对数组的
集合构建对象。这个方法执行与 Object.entries()方法相反的操作

trimStart() 和 trimEnd()

分别用于删除字符串开头和末尾的空格,取代之前的 trimLeft()和 trimRight()
