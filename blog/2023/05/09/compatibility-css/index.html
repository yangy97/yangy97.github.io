CSS问题

padding

如果容器可以滚动 在IE和FireFox下 ``padding-bottom会被忽略,chrome不会

chrome是子元素超过content box触发滚动条显示,在IE和FireFox,是padding box会触发滚动条

建议: 页面底部留白不建议用padding

margin 合并规则

正正取最大,正负值相加,负负最负值

margin=>auto

auto是平分剩余空间

border绘制与color变色

eg:上传图片按钮 hover后变色

可以利用border不设置颜色,直接修改color来实现

透明border的应用

优雅增加点击区域大小

输入框删除按钮 移动端太小 不容易点击

加透明border

绘制三角形

同理绘制梯形就很简单

只需要把其他边框透明

同理将宽度变为 0 又可以绘制三角形
那么各式各样的都能制作 比如气泡箭头等

x-height

x-height表示小写字母x等高度,也就是基线和等分线之间的距离
vertical-align:middle,在css中middle表示基线往上1/2 x-height的高度

line-hieght

可以看出 纯内联元素 可视高度由line-height决定

line-height

如果是数字 所有子元素继承这个数字
如果是百分比或者长度 那么子元素继承 计算后的值
